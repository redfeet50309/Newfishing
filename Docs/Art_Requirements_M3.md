# Milestone 3 美術立繪與素材需求清單 (Art Requirements)

**致：主美術 (Lead Artist) 與 技術美術 (TA)**
**目標：Milestone 3 (五大區域、船隻系統與 30 種魚圖鑑)**
**風格規範：可愛扁平化 / 向量風格 (Cute Flat / Vector Art)**

為了讓前端程式能夠順利實裝 Milestone 3 的全新內容，請美術團隊協助產出以下素材清單。請注意：所有素材必須遵守「無邊框 (No Outlines)」、「簡約幾何 (Minimalist)」、「圓潤邊角 (Rounded Edges)」與「高彩度柔和色調 (Pastel Colors)」等扁平化核心原則。您可以隨時下達 `/flat_art_generator` 指令來獲得靈感 Prompt。

---

## 🖼️ 1. 漁場背景圖 (Location Backgrounds)
**規格：** `1920x1080` (配合 H5 RWD 縮放) 的場景原畫，或按「前景、中景(水面)、背景(天空)」拆分圖層以實作視差滾動 (Parallax)。
**數量：5 張**
1. **恬靜小溪 (Tranquil Stream)**：陽光明媚的森林溪流，水面閃爍著光芒，有些許鵝卵石與青草。
2. **近海防波堤 (Coastal Breakwater)**：看得到遠處的燈塔，消波塊排列在畫面周圍，海浪拍打。
3. **迷霧珊瑚礁 (Misty Coral Reef)**：水質清澈透藍，水面下隱約可見色彩斑斕的珊瑚與熱帶魚剪影。
4. **深海斷層 (Deep Ocean Fault)**：極度漆黑、深藍色調，只有少數發光植物或遠處微弱的光點，充滿壓迫感。
5. **魔幻遺跡 (Mystic Ruins)**：水底發出神秘的紫色/青色螢光，背景有沉沒的希臘羅馬式大理石柱或奇異符文。

---

## 🛥️ 2. 船隻圖標 (Boat Vector Icons)
**規格：** UI 介面使用的 SVG 或高解析去背 PNG。若未來有大張的地圖探索介面，則額外需要去背的 Side View (側面) 向量全圖。
**數量：4 張 (步行不需圖示或只需鞋子圖示)**
1. **橡皮艇 (Rubber Dinghy)**：簡單的橘色充氣艇，配有小馬達。
2. **雙引擎快艇 (Speedboat)**：流線型的白色私人遊艇。
3. **遠洋探勘船 (Trawler)**：重型鋼鐵漁船，有大型起重機與漁網設備。
4. **魔法潛艦 (Magical Submarine)**：融合蒸汽龐克與魔法發光水晶的深海載具。

## 🐟 3. 魚類圖鑑 Icon (Fish Vector Icons)
**規格：** 高解析度 SVG 或 256x256 去背 PNG，必須保持圖示圓潤、乾淨無雜點。
**需求數量：30 張 (分為 5 區，每區 6 隻)**
**設計要點：**
- **輪廓辨識度**：由於圖鑑未解鎖時會直接套用「純黑剪影濾鏡 (Black Silhouette Filter)」，在繪製圖示時，請確保魚的**外觀輪廓差異極大**（例如鯊魚的背鰭、魟魚的扁平、燈籠魚的頭燈）。不要畫一堆輪廓一模一樣、只有顏色不同的魚，這樣剪影看起來會全部一樣。
- **色彩分級 (推薦)**：您可以嘗試在設計上偷偷融入稀有度顏色暗示玩家。例如 Mutant 變異種可以自帶紅色/紫色的邪氣光環。

**魚類清單 (供美術核對品項)：**
*   **小溪**：蝌蚪、鯉魚、鯽魚、鱒魚、雷魚、變異黃金鯉。
*   **近海**：吻仔魚群、竹莢魚、石斑、鱸魚、飛魚、寄居蟹。
*   **珊瑚礁**：小丑魚、河豚、獅子魚、魟魚、旗魚、深海毒觸手。
*   **深海**：盲鰻、燈籠魚、腔棘魚、黑鮪魚、大白鯊、水怪幼體。
*   **遺跡**：幽靈水母、寶箱怪魚、鄧氏魚、星鯨、遠古海龍、克蘇魯之卵。

## 📱 4. UI 介面擴充組件 (UI Elements)
**規格：** Sprite Sheet 或獨立 PNG，切圖需保持透明背景。
1. **地圖選擇介面框架 (Map Selector UI)**：包含進入新區域的「解鎖鎖頭 (Padlock)」、「選擇按鈕」。
2. **圖鑑介面 (Bestiary Book UI)**：一個翻開的書本或科技平板皮套底圖，用來乘載 3x2 或 4x2 的 Icon 網格排列。
3. **稀有度邊框 (Rarity Borders)**：6 種顏色的扁平化圓角邊框，用於套在魚的 Icon 外圍 (白、綠、藍、紫、金、紅)。
